## How to Upload to Google Colab

### Option 1: Manual Upload (Simplest)
1. Open Google Colab: https://colab.research.google.com/
2. Click "Upload" and select `Crisis_Flight_Cancellation_Pipeline.ipynb`
3. Once notebook opens, click the folder icon (📁) in left sidebar
4. Click the upload button and select both CSV files:
   - `gdelt_crisis_aviation_clean.csv`
   - `flight_cancellations_daily_2024.csv`
5. Wait for uploads to complete (~10 seconds for both files)
6. Run cells in order from top to bottom

### Option 2: Code Upload (Alternative)
Add this cell at the top of the notebook and run it:
```python
from google.colab import files
uploaded = files.upload()
# Select both CSV files when prompted
```

## Troubleshooting

### If files not found error:
- Verify all 3 files uploaded successfully
- Check file names match exactly (case-sensitive)
- Re-upload if needed using folder icon in sidebar

### If GPU not available:
- Go to Runtime → Change runtime type
- Select "T4 GPU" under Hardware accelerator
- Click Save

### If model download fails:
- Check internet connection
- Re-run the model download cell
- Colab automatically retries failed downloads
